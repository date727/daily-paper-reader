## Abstract 

Linear attention mechanisms have emerged as promising alternatives to softmax attention, offering linear-time complexity during inference. Recent advances such as Gated DeltaNet (GDN) and Kimi Delta Attention (KDA) have demonstrated that the delta rule an online gradient descent update enables superior associative recall compared to simple additive updates. While KDA refined the coarse head-wise decay gate into channel-wise decay, the learning rate βt in the delta update remains a scalar, limiting the model’s capacity for dimension-specific adaptation. We introduce FG 2-GDN , which replaces the scalar βt with a channel-wise vector analogous to the transition from SGD to per-coordinate adaptive optimizers such as AdaGrad and Adam. We further propose FG 2-GDN+ , which decouples the scaling for keys and values, enabling independent control of erasure strength and write strength. Experiments on synthetic and real-world benchmarks show that FG 2-GDN and its variant improve associative recall and long-context understanding over GDN and KDA, with comparable computational efficiency. 

## 1 Introduction 

The Transformer architecture has revolutionized natural language processing through its powerful attention mechanism, enabling models to capture long-range dependencies effectively. However, the quadratic complexity of softmax attention with respect to sequence length poses significant challenges for both training efficiency and inference scalability, particularly for long-context applications such as agentic and multimodal scenarios. Linear attention mechanisms [ 1] address this limitation by reformulating attention as a linear recur-rence with matrix-valued hidden states, reducing inference complexity from O(L2) to O(L). Early variants of linear attention underperformed softmax attention, but recent innovations have substan-tially narrowed this gap. Two key developments have driven this progress: (1) gating mechanisms that enable adaptive memory decay [ 2– 4], and (2) the delta rule that facilitates precise memory updates through online gradient descent [5, 6]. The delta rule treats the recurrent state as a learnable associative memory that continuously corrects itself toward the target mapping kt 7 → vt. Gated DeltaNet (GDN) [ 4 ] subsequently unified the delta rule with gating mechanisms, introducing a scalar decay factor αt ∈ (0 , 1) that enables controlled memory forgetting. More recently, Kimi Delta Attention (KDA) [ 7] refined GDN’s coarse head-wise decay into channel-wise decay through a diagonal gating matrix Diag (αt), which enables each feature dimension to

2.3 Per-Coordinate Adaptive Learning Rates 

The idea of assigning different learning rates to different parameters has a long history in optimization. AdaGrad [ 8] accumulates squared gradients to scale each coordinate inversely by its historical magnitude, enabling faster progress along infrequent-gradient dimensions. Adam [ 9 ] further combines per-coordinate scaling with momentum, and has become the default optimizer for training deep networks. A common insight underlying these methods is that a single global step size is often suboptimal when different coordinates of the parameter space exhibit different curvatures or update frequencies. Per-coordinate adaptation can yield faster and more stable convergence in such settings. This observation motivates our extension of the delta rule’s scalar learning rate to a channel-wise vector, as described in the next section. 

## 3 Gated Delta Net with Fine-grained Beta 

3.1 Motivation 

As discussed in Section 2.3, per-coordinate adaptive learning rates have proven broadly effective in optimization. A similar asymmetry exists in the gated delta rule: KDA introduced channel-wise forgetting ( αt ∈ Rdk ) but retained a scalar writing rate ( βt ∈ R). This means all dimensions share the same learning rate during each delta update, regardless of their individual relevance to the current context. We propose to close this gap by extending βt to a channel-wise vector as well.

3.6 Hybrid Architecture 

Purely linear attention models are limited by their finite-state capacity [ 17 ]. Following recent hybrid designs [ 4 , 7], we interleave FG 2-GDN layers with Multi-head Latent Attention (MLA) [ 18 ] layers at a 3:1 ratio (Figure 1). The MLA layers handle tasks that benefit from full pairwise interaction, while the linear attention layers provide efficient long-range recurrence. 

## 4 Experiments 

Setup Our experiments evaluate FG 2-GDN alongside FG 2-GDN+, which uses separate channel-wise βkt and βvt for keys and values respectively. We compare against two strong baselines: Gated DeltaNet (GDN) [ 4] with scalar αt and scalar βt, and Kimi Delta Attention (KDA) [ 7] with channel-wise αt ∈ Rdk but scalar βt. For fair comparison, all models share an identical hybrid architecture that interleaves Multi-head Latent Attention [ 18 ] and linear attention layers at a 3:1 ratio. We sample tokens from the SlimPajama dataset and train at two scales: 340M (15B tokens) and 1.3B (100B tokens) using AdamW with 8K sequence length. Full details are shown in Section A.

4.2 Long-Context Evaluation 

We evaluate long-context capability on two benchmarks with the 1.3B models: RULER [ 25 ], which tests 13 retrieval categories at sequence lengths 4k–16k (Table 2), and LongBench (Table 3), which covers long context understanding tasks like document QA, summarization, few-shot learning, and code completion. 

Fine-grained βt sharpens key-value bindings. On single-key retrieval tasks with varied query patterns (MQ, CWE), FG 2-GDN delivers substantial gains over both GDN and KDA. We attribute this to the per-coordinate writing mechanism: by assigning different learning rates to different feature dimensions, the model can concentrate its write energy on the most discriminative dimensions of each key, producing tighter key-value bindings that survive query-time perturbation. This effect is most visible on tasks where the retrieval target is unique, but the query formulation varies. 

Complex retrieval benefits from decoupled writing and forgetting. On multi-key retrieval (MK2), FG 2-GDN+ leads across all sequence lengths, suggesting that decoupling erasure and write strength helps manage multiple competing associations. The advantage is most pronounced at longer contexts where interference between stored key-value pairs intensifies. 

Graceful extrapolation beyond training length. Though all models are trained with an 8K sequence length, we extend the test settings to 16K to further explore the extrapolation ability of NoPE-linear hybrid models. 4.3 Ablation Study 

To disentangle the contributions of our two design choices, namely channel-wise (vector) β and separate βk/β v for keys and values, we train additional 1.3B variants that enable each modification independently. Section 4.3 reports language modeling and LongBench results. 

Channel-wise β improves language modeling. FG 2-GDN (vector β, shared for k and v) achieves the best LM average accuracy and LAMBADA perplexity, outperforming KDA by 0.8 points and reducing perplexity from 13.80 to 13.09. This suggests that per-dimension writing rates are particularly beneficial for short-context language understanding tasks. 

Separate βk/β v improves long-context tasks. The variant with separate scalar βk and βv (with-out vectorization) also improves over KDA on both LM and LongBench metrics, indicating that decoupling erasure and write strength is beneficial even without channel-wise granularity. 

The two improvements are complementary. FG 2-GDN+ combines both modifications and achieves the highest LongBench average (18.3%) and few-shot score (32.1%), substantially outperforming all other variants on long-context tasks. While its LM accuracy (53.4%) is slightly below FG 2-GDN, it provides the best overall trade-off across short- and long-context settings, consistent with its strong performance on RULER (Table 2). 

4.4 Efficiency Expressiveness at near-zero cost. Figure 2 compares the prefill throughput and latency across configurations from 2k ×16 to 32k ×1. At the longest sequence (32k), all linear attention variants achieve 1.7–1.9 × speedup over MLA. The channel-wise βt parameterization introduces less than 5% overhead relative to KDA across all settings, and the gap narrows as sequences grow—precisely the regime where the expressiveness benefits matter most. FG 2-GDN+ incurs a slightly larger overhead from the additional βvt projection but remains competitive with all baselines.

that shares structural similarity with the delta rule but replaces the exponential decay with a normalized rank-1 subtraction. This design also admits an online learning interpretation analogous to DeltaNet’s. 

Delta Rule and Online Learning. The delta rule, also known as the Widrow–Hoff learning rule [31 ], interprets each recurrent update as a single SGD step on the objective ∥S⊤kt − vt∥2. DeltaNet [5, 6 ] introduced this principle to linear attention, achieving superior associative recall through rank-1 corrective updates. Gated DeltaNet (GDN) [ 4] unified the delta rule with a scalar forget gate αt,8and Longhorn [ 32 ] formalized the connection to amortized online learning. KDA [ 7 ] refined the gate from scalar to channel-wise Diag (αt), significantly improving long-context modeling. Our FG 2-GDN extends this line of work along the orthogonal axis of writing granularity , replacing the scalar learning rate βt with a per-coordinate vector βt ∈ Rdk .

Hybrid Architectures. Purely linear models are fundamentally constrained by finite-state capacity, making exact copying and fine-grained long-context retrieval challenging [ 17 ]. Hybrid architectures that interleave linear and full attention layers have emerged as a practical compromise. Jamba [ 33 ] combines Mamba with Transformer blocks in a MoE setting; Gated DeltaNet [ 4 ] explored hybrids with sliding-window and global attention; Kimi Linear [ 7] adopts a 3:1 KDA-to-MLA ratio, demonstrating that such hybrids can surpass full-attention baselines in both quality and efficiency at scale. Our work follows this inter-layer hybrid paradigm, using the same 3:1 ratio to ensure fair comparison, and focuses on improving the linear attention component. 

## 6 Conclusion 

We present FG 2-GDN, an extension of the delta rule-based linear model that replaces the scalar learning rate βt in the delta update with a channel-wise vector, allowing each feature dimension to adapt its writing strength independently. Experiments at the two scales show that this fine-grained control consistently improves long-context retrieval performance on RULER and LongBench while maintaining competitive language modeling quality. The additional parameter overhead is marginal, and prefill throughput remains comparable to the baseline GDN variant. We hope this work provides a useful data point on the benefits of fine-grained adaptivity in linear recurrent models.

## A Experimental Details 

All models in our experiments share an identical hybrid architecture that interleaves Multi-head Latent Attention (MLA) [ 18 ] layers with linear attention layers in a 3:1 ratio (i.e., three MLA layers followed by one linear attention layer). This design follows the inter-layer hybrid paradigm adopted by Kimi Linear [ 7 ] and ensures that the only variable across compared models is the linear attention mechanism itself. We experiment at two scales: 340M parameters (20 layers, hidden size 1024) and 1.3B parameters (24 layers, hidden size 2048). Both MLA and linear attention use 16 heads. The MLA layers adopt low-rank projections for queries and key-value pairs (q_lora_rank and kv_lora_rank in Table 6), and we disable RoPE in MLA layers so that the linear attention layers can serve as the sole source of positional information through their recurrent dynamics, following the design choice in KDA. The linear attention head dimension is fixed at 128 across both scales. Both scales are trained with the AdamW optimizer [ 9] using a cosine annealing schedule with linear warmup. The peak learning rate is 3 × 10 −4, decaying to 3 × 10 −5. The 340M model is trained on 15B tokens with 0.5B warmup tokens and a per-step batch size of 0.5M tokens; the 1.3B model is trained on 100B tokens with 1B warmup tokens and 2M tokens per step. All training uses a sequence length of 8192 and BF16 mixed precision.

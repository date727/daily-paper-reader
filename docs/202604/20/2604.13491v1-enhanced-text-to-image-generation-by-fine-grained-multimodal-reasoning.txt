## Abstract 

With the rapid progress of Multimodal Large Language Models (MLLMs), unified MLLMs that jointly perform image understanding and generation have advanced significantly. How-ever, despite the inherent reasoning capabilities of unified MLLMs for self-reflection and self-refinement, their use in text-to-image generation remains largely underexplored. Meanwhile, ex-isting multimodal reasoning–based image genera-tion methods mostly rely on holistic image–text alignment judgments, without fine-grained reflec-tion and refinement of detailed prompt attributes, leading to limited fine-grained control. Therefore, we propose Fi ne-grained Multimodal Reasoning (FiMR ), a framework that leverages decomposed visual question answering (VQA) to break down an input prompt into minimal semantic units— such as entities and attributes—and verify each unit via VQA to generate explicit, fine-grained feedback. Based on this feedback, FiMR then applies targeted, localized refinements. This fine-grained self-reasoning and self-refinement enable MLLMs to achieve more precise improvements in image–prompt alignment and overall generation quality at test time. Extensive experiments demon-strate that FiMR consistently outperforms image generation baselines, including reasoning-based methods, particularly on compositional text-to-image benchmarks.

In the realm of image understanding, reasoning techniques (Bai et al., 2025; Wang et al., 2025; Xu et al., 2025) have success-fully addressed a key limitation of traditional autoregressive (AR) models—once generated, previous outputs cannot be revised, by trading increased inference-time computation for progressive improvements of image comprehension (Bai et al., 2025; Wang et al., 2025). However, for the task of text-to-image (T2I) generation, reasoning-based approaches for improving image–prompt alignment and image generation quality remain underex-plored. Early research on image reasoning focused mainly on Chain of Thought (CoT) strategies such as prompt aug-mentation (Jiang et al., 2025; Zhang et al., 2025). By ex-panding simple prompts into dense and structured descrip-tions, these methods provide a T2I model with richer context to facilitate better initial synthesis. Nevertheless, these ap-proaches cannot revise the generated image, even when it is misaligned with the prompt. To address this limitation, a recent work (Pan et al., 2025b) has introduced regenera-tion cycles. It self-evaluates image–prompt alignment for self-generated images and iteratively regenerates the image when misalignment is detected. However, it largely relies on holistic image–prompt alignment evaluation, which makes it difficult to identify and refine specific missing or incorrect prompt details. To address these limitations, we propose Fi ne-grained 

Multimodal Reasoning ( FiMR ), a MLLM-based framework that generates explicit fine-grained feedback via Decom-posed Visual Question Answering (VQA) to achieve precise prompt-image alignment and enhanced generation quality. The framework consists of three main steps: (1) Initial Text-to-Image Generation, (2) Fine-grained Feedback Generation, and (3) Localized Image Correction. 

In the Initial Text-to-Image Generation step, the reasoning MLLM first generates an image from the input prompt. Then, in the Fine-grained Feedback Generation step, the input prompt is decomposed into minimal semantic units, such as entities, attributes, and relations. The generated units are then separately evaluated via self-VQA by the MLLM to assess the fine-grained alignment between the prompt and the generated image. When misalignments are 1

## 2. Related Work 

Unified MLLMs Recent unified MLLMs increasingly integrate image understanding and generation within a sin-gle architecture. These models are primarily categorized into two streams. One line of research adopts discrete vi-sual tokens to frame image generation as an autoregressive next-token prediction task, modeling both text and images with a unified cross-entropy loss (Wu et al., 2024; Wang et al., 2024). In contrast, another stream employs continuous visual features for image synthesis by coupling the trans-former with diffusion or flow-based processes (Xie et al., 2025; Ma et al., 2025b).

Recent research has begun to explore reasoning capabili-ties within unified MLLM-based image generation. While T2I-R1 (Jiang et al., 2025), ReasonGen-R1 (Zhang et al., 2025), and GoT-R1 (Duan et al., 2025) enhance brief user instructions by transforming them into enriched contexts for reasoning, they lack post-generation correction mechanisms, leaving initial semantic misalignments unaddressed. Conse-quently, Janus-Pro-R1 (Pan et al., 2025b) overcomes this by leveraging unified MLLMs to assess visual alignment and trigger iterative regeneration using their dual understanding and generation capabilities. Despite its effectiveness, Janus-Pro-R1 relies on coarse global assessments without fine-grained localization, lead-ing to unnecessary full-image regeneration for otherwise minor errors. In contrast, our approach decomposes prompts at a finer semantic level to enable localized correction; this allows the model to selectively modify misaligned regions while preserving correct areas, significantly enhancing both precision and efficiency.

3.1. Limitation of Global Judgment 

We generated images from GenEval prompts (Ghosh et al., 2023) using Janus-Pro-R1 (Pan et al., 2025b), following the reasoning process shown in Figure 1, and used GenEval’s automated protocol to assign each image a binary alignment label (Aligned/Misaligned), which we treat as a ground-truth (GT) label. The protocol verifies object presence/-counting/relations via detector-derived bounding boxes and checks color attributes via CLIP similarity on cropped ob-ject regions. When evaluated against this GT, we observe that Janus-Pro-R1’s global judge exhibits significant mis-alignments, as it fails to accurately identify fine-grained semantic mismatches. As shown in Table 1, global judgment is prone to false ac-ceptance, particularly in categories requiring fine-grained precision. Specifically, the model exhibits high False Posi-tive (FP) rates—reaching 32.50% for counting and 23.50% for color attributes—where misaligned images are incor-rectly judged as aligned. With an overall FP rate of 14.21%, the model frequently fails to detect localized semantic dis-crepancies, indicating that coarse global assessment is in-sufficient for precise image–prompt alignment verification. Detailed experimental procedures and methodologies are provided in Appendix A. 

3.2. Rationale Bypass Problem 

Beyond global judgment errors, we identify the Rationale Bypass Problem, where models struggle to effectively uti-lize self-reflection results for practical self-refinement. To test whether regeneration truly depends on the rationale con-tent, we perturb the key semantics of the rationale shown in Figure 1-for example, changing “green motorcycle” to “yellow motorcycle with a bench”-while keeping the original prompt unchanged. Detailed experimental procedures and methodologies are provided in Appendix A. As shown in Table 2, image alignment improves even when the rationale is semantically perturbed. This indicates that the model does not meaningfully use the rationale content to guide generation. Consequently, these findings underscore the necessity of explicit feedback. 

## 4. Method 

4.1. FiMR: Multi-step multimodal reasoning framework 

Based on the motivation in Section 3, we propose the Fine-grained Multimodal Reasoning (FiMR) framework. For a given prompt, FiMR evaluates prompt-image alignment at a fine-grained level by utilizing a decomposed VQA, pro-vides explicit feedback when alignment fails, and iteratively corrects the image based on the feedback. Our framework follows a systematic three-step self-reasoning procedure: (1) Initial Text-to-Image Generation, (2) Fine-grained Feedback Generation, and (3) Localized Image Correction. Figure 2 provides an overview of the FiMR. 4.1.1. S TEP 1: I NITIAL TEXT -TO -I MAGE GENERATION 

In Step 1, the MLLM performs initial T2I generation for a given input prompt P , following the same procedure as conventional T2I models. The primary objective of this step is to generate images I that are well-aligned with the prompt while satisfying the specified constraints. 3Submission and Formatting Instructions for ICML 2026 

Figure 2. Overview of FiMR. The framework iteratively refines the alignment between images and prompts through three steps. Step 1: Initial Text-to-Image Generation involves the initial synthesis of an image from a complex input prompt using the generation capability of a Unified MLLM. Step 2: Fine-grained Feedback Generation utilizes the model’s understanding capacity to conduct a fine-grained evaluation of the generated image. As shown on the right, this stage is composed of four sub-processes: summarizing the long prompt, decomposing it into multiple semantic units via Decomposed VQA, verifying each individual tuple against the visual content to identify specific mismatches, and generating explicit feedback instructions. Finally, in Step 3: Localized Image Correction , the model executes localized edits based on the provided feedback to rectify specific erroneous regions while preserving the rest of the image. This entire loop is repeated until the image-prompt alignment is fully satisfied. 

4.1.2. STEP 2: F INE -GRAINED FEEDBACK GENERATION 

Step 2 is the self-reflection stage within our unified frame-work, where the model performs fine-grained alignment judgment and feedback generation. In this stage, the frame-work takes the input prompt P and the image I as inputs to perform decomposed VQA, subsequently generating fine-grained feedback. Given this input, the model gen-erates a reasoning path Φ = {Psummary , T , V, F }. Specifi-cally, the model first derives a summarized prompt Psummary 

that removes subjective or non-verifiable details to estab-lish a stable evaluation basis. It then decomposes Psummary 

into a set of semantic tuples (e.g. whole-entity(oven), attribute-color(oven,pink)) T = {T i}Ni=1 , covering enti-ties, attributes, relationships, and special categories such as counting and text, following prior structured grounding paradigms. For each tuple Ti, the model performs image-conditioned verification, generating a rationale and a binary alignment decision, collectively denoted as V. Each ele-ment in V = {V i}Ni=1 consists of an explicit reasoning trace paired with a binary verdict (“Yes” or “No”) indicating whether the corresponding tuple is satisfied by the image. Finally, based on the verification outcomes, the model syn-thesizes feedback F (·): for misaligned images, F (Imisalign )

consolidates the failed rationales into explicit corrective in-structions, whereas for aligned images F (Ialign ), it emits a termination signal indicating no further refinement is re-quired. 4.1.3. S TEP 3: L OCALIZED IMAGE CORRECTION 

Step 3 constitutes the self-correction phase of the FiMR framework, where fine-grained feedback is utilized to per-form localized image correction. When the generated image is considered aligned with the prompt P , denoted as Ialign ,the framework produces a termination signal, thereby clos-ing the iterative correction loop. In contrast, when the generated image is judged to be mis-aligned with the prompt, denoted as Imisalign , the framework leverages the corrective feedback F (Imisalign ) as an explicit guidance signal. This feedback specifies which provides ac-tionable instructions for resolving the identified mismatches, enabling the model to perform localized and intent-aware image correction. The resulting image from Step 3 is then re-processed as in Step 2, forming an iterative loop that continues until the alignment is resolved. 4Submission and Formatting Instructions for ICML 2026

## 5. Experiments 

Training Datasets For training, we utilize a total of 200K image-text pairs. This collection consists of 140K sam-ples from FocusDiff (Pan et al., 2025a) for the edit dataset and 60K samples from our composition-specific synthetic dataset. The synthetic data are generated from prompts adapted from prior work (Liu et al., 2025; Zhang et al., 2025) via Qwen-Image and Qwen-Image-Edit (Wu et al., 2025). All data are processed into the reasoning data samples as described in Section 4.2.1. Details of training datasets are provided in Appendix B. 

Baseline Models To ensure a fair and rigorous evaluation, we employ Janus-Pro-7B (Chen et al., 2025) as the primary backbone for both our proposed FiMR framework and the baseline models. We compare FiMR against two represen-tative reasoning-based methods: Janus-Pro-R1 (Pan et al., 2025b) and T2I-R1 (Jiang et al., 2025). While all models share the same underlying architecture, the baselines differ fundamentally in their reasoning granularity. Janus-Pro-R1 operates on a global level, performing self-judgment and full-image regeneration, whereas T2I-R1 focuses on text-centric Chain-of-Thought (CoT) prompt augmentation prior to generation. By standardizing the backbone and reproduc-ing all results in an identical environment, we isolate the effectiveness of our fine-grained image reasoning approach. Additionally, beyond the 7B model, we conduct further ex-periments using Janus-Pro-1B to validate the scalability of our framework. 

Benchmarks To assess the image reasoning capabilities of our models, we utilize three comprehensive benchmarks: DPGBench (Hu et al., 2024), GenEval (Ghosh et al., 2023), and T2I-CompBench (Huang et al., 2023). For DPGBench, we found issues in the official evaluation code under multi-GPU configurations. To ensure a reliable assessment, we employed a revised version with the corrections. For a fair comparison, we re-evaluated all compared image reasoning models under the same corrected setup. For GenEval and T2I-Compbench, we strictly follow the official implemen-tations to ensure consistency and comparability. Detailed descriptions of each benchmark are provided in Appendix C. 

Implementation Details Our models were trained on eight NVIDIA A100 GPUs. For a detailed list of hyper-parameter settings, including learning rates and batch sizes, please refer to the Appendix D.

## 6. Ablation Studies 

To provide a deeper understanding of the individual contri-butions within our framework, we deconstruct the proposed framework and evaluate its core components. The following ablation studies are designed to isolate the impact of fine-grained judgment strategy, the necessity of explicit feedback for image correction, and the robustness of the framework across varying model scales. 

6.1. Effect of Global Judge vs. Fine-grained Judge 

To investigate the impact of judgment granularity on image-prompt alignment, we compare the performance of a Global Judge (holistic evaluation) and our Fine-grained Judge (De-composed VQA). The model configurations and experimen-tal procedures follow the same methodology described in Section 3.1. As shown in Table 1, the decomposed VQA approach sig-nificantly reduces the False Positive (FP) rates across nearly all categories compared to the Global Judge. This reduction is most pronounced in complex compositional tasks; for in-stance, in the Count category, the FP rate drops dramatically from 32.50% to 15.94%, and the Overall FP rate decreases from 14.21% to 10.60%. These results indicate that a Global Judge often fails to detect subtle misalignments. By contrast, our decomposed VQA framework verifies the image against minimal semantic units independently.

## A. Details of Preliminary Experiments 

In this section, we provide the comprehensive experimental procedures and methodologies employed in the diagnostic studies presented in Section 3. To validate the necessity of our proposed framework, we detail the ground truth labeling process, the specific implementation of the judgment paradigms (Global vs. Fine-grained), and the setup for the rationale perturbation experiment. These details ensure the reproducibility of our findings regarding judgment inaccuracies and the rationale bypass problem in current reasoning models. 

A.1. GenEval Evaluation Protocol as Ground Truth 

To establish a reliable gold standard for image-prompt alignment, we utilize the automated evaluation protocol of the GenEval (Ghosh et al., 2023) as our ground truth (GT). The protocol systematically assesses alignment across six categories, utilizing specific vision models to ensure objective and reproducible verification. The detailed mechanisms for each category are as follows: • Single and Two Object Alignment : The protocol employs a pre-trained object detector to extract bounding boxes (Bboxes) from the generated image. Alignment is determined by verifying whether the Bboxes corresponding to the objects specified in the prompt are successfully detected. • Counting and Position : Building upon the initial detection phase, these categories involve higher-level logic. For 

Counting , the protocol verifies the exact number of detected Bboxes against the prompt. For Position , it evaluates the spatial relations (e.g., “left of”, “above”) between the extracted Bboxes to confirm semantic alignment. • Colors and Color Attributes : To verify color-related constraints, the protocol first crops the relevant object region using its detected Bbox. It then utilizes the CLIP (Radford et al., 2021) model to calculate the similarity between the cropped image region and the color attribute specified in the prompt, issuing an “Align” label only if the similarity exceeds a predefined threshold. 

A.2. Implementation of Global Judge 

The Global Judge follows the native prompt-image alignment evaluation process for which Janus-Pro-R1 was trained. After generating an image, the model is prompted to evaluate the alignment between the generated image and the prompt at a holistic level. Specifically, we append the trigger phrase, “Let me think. Does this image match the prompt...” to initiate the judgment process. The model is trained to provide a binary answer (“Yes” or “No”) followed by a concise explanation of one to two sentences. This approach forces the model to perceive the entire image and the complete prompt as single entities, often leading to the high False Positive rates observed in our experiments due to the omission of subtle details. 

A.3. Implementation of Rationale Perturbation Experiment 

To verify the Rationale Bypass Problem discussed in Section 3, we designed a structured perturbation pipeline to decouple the semantic content of the feedback from the regeneration trigger. The experiment follows a three-stage process: 1. Initial Generation and Reflection : We first generate an image using Janus-Pro-R1 for a given GenEval prompt. Following the generation, the model enters a reflection phase where we extract its internal Self-Judge (Yes/No) and 

Self-Feedback (Rationale) logs. Since only samples judged as “No” by the internal Self-Judge are eligible for the subsequent regeneration phase, we specifically selected these samples to perform our perturbation experiment. These logs represent the model’s native, unperturbed reasoning regarding the alignment of the first iteration. 2. Automated Perturbation via Qwen3-32B : To systematically corrupt the semantic details of the feedback, we utilize the Qwen3-32B (Yang et al., 2025) model. From the initial pool of 2,212 generated images, we identified 368 samples that were targeted for regeneration based on the model’s internal “No” judgments. Consequently, we applied our perturbation process to the feedback of these 368 specific samples. We provide Qwen3 with a system prompt and specific few-shot examples designed to alter key entities, colors, or spatial relations within the original rationale while maintaining its overall linguistic structure. This ensures that the model receives a plausible but factually incorrect rationale as input for the next step. 11 Submission and Formatting Instructions for ICML 2026 

3. Regeneration with Perturbed Feedback : Finally, we perform the regeneration process by feeding the perturbed feedback back into Janus-Pro-R1. The model is then tasked with generating a refined image based on the corrupted reasoning. As detailed in Section 3, the experimental results show that the model’s alignment performance improved from the first iteration even when the feedback was semantically corrupted. The performance gain in the perturbed case was nearly identical to that of the unperturbed case, confirming that the model utilizes the feedback as a binary signal for global regeneration rather than as a semantic guide for localized correction.

B.3. Dataset Construction Pipeline 

In this section, we provide a comprehensive description of the multi-stage pipeline used for dataset construction and fine-grained judgment. As previously noted, we utilize Qwen3-Next-80B-A3B (Yang et al., 2025) for core text-based tasks-including prompt summarization, tuple decomposition, and question/feedback generation-and Qwen-VL-32B (Bai et al., 2025) to perform the Visual Question Answering (VQA) required for judgment. P ROMPT SUMMARIZATION (PSUMMARY )To capture the core intent of long or complex input prompts, we perform a summarization step. The model is prompted to extract the essential semantic elements from the raw input prompt, resulting in a condensed representation Psummary . This ensures that subsequent decomposition steps focus on the primary subjects and attributes. B.3.2. DSG T UPLE DECOMPOSITION AND REFINEMENT 

The construction of the fine-grained tuple set T is the cornerstone of our judgment and feedback mechanism. To ensure the accuracy and schema-compliance of these atomic units, we employ a rigorous three-stage refinement process: 1. Initial Decomposition (Prompt to DSG Tuple) : The initial step involves decomposing the Psum into a set of atomic semantic units following the Davidsonian Scene Graph (DSG) (Cho et al., 2023) format. This represents the prompt as a collection of non-divisible attributes and relations. 2. Tuple Normalization (DSG Tuple Rewrite) : Raw outputs from LLMs often include undefined schemas or inconsistent formats. In this stage, we prompt the model to normalize the initial tuples into the specific categories and structures defined in Section 4.2.1. This ensures that the generated tuples are compatible with our fine-grained judge’s logic. 3. Hallucination Filtering (DSG Tuple Prune) : A known issue in LLM-based DSG decomposition is the generation of implicit information not present in the original text (e.g., assuming a “next to” relation between two mentioned objects). To mitigate this, we perform a pruning step that strictly removes any tuples containing inferred or implicit spatial/semantic relations, retaining only the information explicitly stated in the input prompt. Following this refinement, the final tuple set T is converted into a series of verification questions using the few-shot prompting strategy provided by the original DSG framework. These questions serve as the input for the VQA-based alignment check performed by Qwen-VL-32B (Bai et al., 2025). B.3.3. F EEDBACK GENERATION 

Following the VQA-based alignment check, the resulting verification set V (I) serves as the primary input for the final feedback generation stage. The Qwen3-Next-80B-A3B LLM (Yang et al., 2025) identifies all tuples assigned a “No” judgment-indicating a misalignment-and translates these discrepancies into a structured sequence of edit instructions. Each feedback item is formulated as a step-by-step action (e.g., Add, Change, Reposition ) specifically designed to transform the misaligned elements from “No” to “Yes”. ## C. Benchmark Details 

C.1. GenEval 

GenEval(Ghosh et al., 2023) evaluates object-focused image generation performance using 550 text prompts. These prompts are systematically structured into six primary categories to measure fine-grained compositional adherence: Single Object ,

C.2. T2ICompbench

Compared to prior benchmarks such as T2I-CompBench(Huang et al., 2023), DPGBench(Hu et al., 2024) contains far more distinct nouns (4,286 vs. about 1,400) and significantly longer prompts, enabling fine-grained automatic evaluation of dense prompt
